## Description
# jquery-bootstrap-mmenu
Jquery-bootstrap version of menus from https://mmenujs.com/

## Installation
### bower
`bower install https://github.com/FCOO/jquery-bootstrap-mmenu.git --save`

## Demo
http://FCOO.github.io/jquery-bootstrap-mmenu/demo/
